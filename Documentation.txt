Вот краткая документация с примерами запуска сервиса и вызова его API.

После того, как вы склонируете репозиторий и перейдете в папку проекта, вы можете установить все зависимости, выполнив следующую команду:

pip install -r requirements.txt

Затем, чтобы запустить сервер, выполните:

python manage.py
python manage.py runserver

Эта команда запустит сервер на порту 8000 локальной машины.

Теперь вы можете начать отправлять запросы к API сервиса. Вот некоторые примеры использования:

Зарегистрировать нового пользователя:

POST /users/
{
  "username": "test_user"
}

Получить список всех пользователей:

GET /users/

Отправить заявку в друзья другому пользователю:

POST /friend_requests/
{
  "sender_id": 1,
  "receiver_id": 2
}

Получить список исходящих заявок в друзья для пользователя:

GET /users/1/friend_requests/outgoing/

Получить список входящих заявок в друзья для пользователя:

GET /users/2/friend_requests/incoming/

Принять заявку от другого пользователя:

PUT /friend_requests/1?response=accept

Получить список друзей пользователя:

GET /users/1/friends/

Удалить друга из списка друзей:

DELETE /friends/2/

Получить статус дружбы с другим пользователем:

GET /friend_status?user1_id=1&user2_id=2
